# Jelly

Hi，there！

Jelly是一个MacOS平台的翻译助手，旨在为你带来沉浸式的阅读、写作体验。

Jelly支持划词翻译、截屏翻译、OCR，还有多种AI功能，如润色、语法纠错等。

本仓库用于收集用户反馈、需求文档，以及开源部分提示词。

更多信息请访问 [官方网站](https://jellytranslate.com) （网站建设中）
